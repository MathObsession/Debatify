# Debatify
Debatify is a project that came from one of my friend's hobbies; debating. Due to him having no one to debate with, I planned to create debatify, which is a unified experience for users to debate against AI bots in realtime locally for free! This experience caters not only the professional end of debaters but also the casual ones too. 
This project was made with the ambition that every lonely debater has someone to debate with and fullfil their hobbying journey.
